# pico-8
A mono repo for all my pico 8 games.


## Space Escape
Face off against an onslaught of alien crafts. How high can you score?

[Click here](https://www.lexaloffle.com/bbs/?tid=146778) to play!

**PC**  
- **Laser** = **Z**
- **Move** = Arrow keys

**Mobile**  
- **Laser** = **O**
- **Move** = Directional pad
